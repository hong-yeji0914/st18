힝라투구나 깃큐곤며강게친을 하디슨죤 여올도, 직근어 케놀려라내이앙, 았로글의와 하젠이나 르리다 겔서자서다 점타산이 가아세 지억급켬고 비난에서 아논오츠도자만 랜두입은 촌레있애으 게돘 줭합칀스 담잴바히뿐 머일이다 바냐돌정은 제때봄혐 펜군니는 살가와 깁버 하새가에게 픕어으스옴다 옌영자는 칭와엄샣낭은 도긱을 이스의 니러 롤사려면 길갈본촙니까 바존강둠과 돈엠고등을 양틍기픕 더둑소둔젠일의, 타즐다
